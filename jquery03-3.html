first()

같은 이름의 태그 중에 가장 먼저 나온 태그 선택

여러가지 태그 중 첫번째 h1 태그

화면에 출력될 p 태그

선택할 p 태그

선택할 p 태그 2번째


선택 되지 않을 첫번째 태그인 p 태그

선택 되지 않을 두번째 태그인 p 태그


선택 되지 않을 세번째 태그인 p 태그

선택 되지 않을 네번째 태그인 p 태그



last()

같은 이름의 태그 중에 제일 마지막 태그 선택

여러가지 태그 중 첫번째 h1 태그

선택할 p 태그

선택할 p 태그 2번째


선택 되지 않을 첫번째 태그인 p 태그

선택 되지 않을 두번째 태그인 p 태그


선택 되지 않을 세번째 태그인 p 태그

선택 되지 않을 네번째 태그인 p 태그



eq() -> 같은 태그 중에 특정 순서에 위치한 태그를 선택

filter()

지정한 조건을 만족하면 선택 -> 조건 : class="p1"

4 p tag



not()

지정한 조건을 제외하고 선택 -> 조건 : class="p1"
